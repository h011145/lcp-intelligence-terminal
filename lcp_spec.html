01. PROTOCOL VARIANTS (亜種コレクション)

LCP-IPC: Inter-Process Communication

目的: プロセス間通信の意志的直接執行

mkdir -p /tmp/lcp_ipc/{in,out,err}
# sender.lcp
意志: "データをプロセスBに送信" -> 執行: echo "DATA" | cat > /tmp/lcp_ipc/in/pipe
LCP-NET: Network Protocol

目的: ネットワーク通信の意志的直接描画

# server.lcp
意志: "PORT 8080で待機" -> 執行: nc -l -p 8080 -q 1
LCP-FS: Filesystem Protocol

目的: ファイルシステム操作の意志的直接執行

# watcher.lcp
意志: "ディレクトリ監視" -> 執行: inotifywait -m ~/lcp_fs/incoming
LCP-DB: Database Protocol

目的: データベース操作の意志的直接執行

# insert.lcp
意志: "レコード追加" -> 執行: sqlite3 ~/lcp_db/data.db "INSERT INTO..."
LCP-CRON: Scheduled Protocol

目的: 定時実行の意志を直接cronに描画

意志: "AM 3:00 BACKUP" -> 執行: (crontab -l; echo "0 3 * * * ...") | crontab -
LCP-GIT: Version Control Protocol

目的: Git操作の意志的直接執行

意志: "AUTO PUSH" -> 執行: git add . && git commit && git push
